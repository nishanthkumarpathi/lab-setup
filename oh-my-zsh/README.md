**Step 2: Run the Script.**

**Step 3: Configure the zshrc Profile**

Edit ~/.zshrc file, find plugins=(git) replace plugins=(git) with :

```bash
plugins=(git zsh-autosuggestions zsh-syntax-highlighting)
```
